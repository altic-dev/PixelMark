# PixelMark

PixelMark is a powerful Mac application designed for screen recording and annotation.

## Features

- **Screen Recording**: Capture your screen with high quality.
- **Annotation**: Draw and mark up your screen in real-time.
- **Window Selection**: Easily select specific windows to record.
- **New Online**: Now available and ready to use.

## Getting Started

1. Clone the repository.
2. Open `PixelMark.xcodeproj` in Xcode.
3. Build and run the project.

## Requirements

- macOS 12.0 or later
- Xcode 14.0 or later
